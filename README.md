### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 个人信息、3. 客户信息管理、4. 项目信息管理、5. 项目收益管理、

管理员功能：1. 登录、2. 个人中心、3. 员工管理、4. 客户信息管理、5. 行业类型管理、6. 项目信息管理、7. 项目类型管理、8. 项目收益管理、
